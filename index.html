HiStyleとは？

さぶうぇい製のサイトで使われているCSSをより汎用性の高いものへ改良したCSSフレームワークです。
クラスを極力使用せず、また使用するクラス全てにhs-の接頭辞を採用しています。

フォント

通常の英数字、半角記号などは Jost 、その他文字は Noto Sans CJK jp を採用しています。
また<code>タグ内では Fira Mono 、アイコンフォントとして Material Icons を採用しています。

Jost

HiStyle was inspired by HiSubway.

Noto Sans CJK jp

ハイスタイルはさぶうぇいに触発されました。

Fira Mono

const HiStyle = inspire('HiSubway');

グリッド

Web Gridを採用しているため簡潔でわかりやすくレスポンシブな配置が可能です。
HiStyle本体にグリッド機能は内蔵されていないので好みのフレームワークに差し替えることも可能です。
